# Contalink test frontend

## How to run locally

Clone the repo:
```
git clone git@github.com:omezaldama/contalink-test-fe.git
```
Cd into the repo directory and install dependencies:
```
cd contalink-test-fe
npm install
```
Copy the environment variables file:
```
cp .env.example .env
```
Run the project:
```
npm run dev
```
